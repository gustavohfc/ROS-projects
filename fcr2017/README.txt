Roteiro 5


Dependências:
O pacote 'map_server' é utilizado para salvar a imagem do mapa.

Executando:
Para executar a simulação é necessário apenas o comando 'roslaunch fcr2017 roteiro5.launch'

Saída:
O mapa é mostrado em tempo real no RViz e a imagem final do mapa é gravado no arquivo 'fcr2017/map_output/cic.pgm'
